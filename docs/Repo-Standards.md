# Repo Standards

---

# Project Structure

- Organized by type (`components/`, `utils/`, `types/`, `data/`)
- No dead code
- No unused files

---

# Naming

## Files

- React components: `PascalCase.tsx` (e.g., `BudgetForm.tsx`, `BudgetItemList.tsx`)
- Utilities and data: `camelCase.ts` (e.g., `budgetUtils.ts`, `mockData.ts`)
- Type definition files: `camelCase.ts` (e.g., `budget.ts`)

## Components

PascalCase (matches filename)

## Variables and functions

camelCase
